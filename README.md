# BISINDO Sign Language Detection

Sistem deteksi Bahasa Isyarat Indonesia (BISINDO) berbasis Computer Vision, MediaPipe, TensorFlow, dan Flask.

## Dataset

Dataset yang digunakan dalam proyek ini berasal dari Kaggle:

https://www.kaggle.com/datasets/meisyavira/abjad-bahasa-isyarat-indonesia-bisindo

Atau klik link berikut:

[Download Dataset BISINDO](https://www.kaggle.com/datasets/meisyavira/abjad-bahasa-isyarat-indonesia-bisindo)

## Teknologi yang Digunakan

- Python
- OpenCV
- MediaPipe
- TensorFlow
- Keras
- Flask
